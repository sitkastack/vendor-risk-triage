# Phases

This folder holds per-phase design notes, scope statements, and retrospectives. Each phase of the project (Phase 0 through Phase 6, as outlined in the root README) lands as its own subfolder or document here, with the design decisions and trade-offs recorded alongside the code that implements them.

The intent is that someone reading the repo six months from now can reconstruct *why* each phase landed the way it did — not just *what* was built.
